# Friend Recommender

A Spring Boot application that recommends friends based on mutual connections and location.

## Features
- Add users to the network
- Create friendships between users
- Get friend recommendations based on mutual friends
- Beautiful Tailwind CSS UI
- Delete users from the network

## Running Locally

```bash
./mvnw spring-boot:run
```

Visit http://localhost:8080

## Tech Stack
- Java 17
- Spring Boot 3.2.0
- Tailwind CSS
- Maven
